# 示例原始数据

| 文件 | 格式 |
|---|---|
| `irregular_vocab_01.txt` | 混合：破折号、缩进、冒号 |
| `irregular_vocab_02.txt` | 制表符多列 |
| `irregular_vocab_03.txt` | 编号列表 |
| `irregular_vocab_04.txt` | 缩进层级 |
| `vocabulary_01.txt` | 制表符 2 列 |
| `vocabulary_02.txt` | 制表符 2 列 |
