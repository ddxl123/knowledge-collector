# 版权合规指南

## 版权风险等级

**所有等级均可收集，根据风险等级向用户发出相应提示后继续。**

### 高风险（需强烈提示）
- 完整复制受版权保护的教材、书籍
- 未经授权的商业学习资料
- 受保护的考试真题（官方考试题库）
- 付费课程内容

→ ⚠️ 强烈提示版权风险，明确告知仅供个人学习、不得传播或商用，用户确认后继续

### 中风险（需提示）
- 网站原创内容（需查看使用条款）
- 用户生成内容（需确认授权）
- 部分开放许可内容

→ 提示用户注意使用条款，建议仅供个人学习

### 低风险（直接收集）
- 公共领域内容（版权过期）
- 开放许可内容（CC0、CC-BY、CC-BY-SA）
- 官方公开发布的考试大纲
- 事实性数据（公式、历史年表）

→ 无需提示，直接收集

## 合规策略

### 1. 优先使用开放许可资源
- Creative Commons 许可（CC0、CC-BY、CC-BY-SA）
- 公共领域作品（作者去世 50 年以上）
- 政府公开数据（考试大纲、课程标准）

### 2. 内容改写与原创
- 提取核心概念，用自己的话重新表述
- 增加原创解释、示例、练习题
- 改变内容组织方式
- 综合多个来源创建新内容

### 3. 来源标注
每条收集的内容必须记录：
- URL 或文件路径
- 作者（如有）
- 出版/访问日期
- 许可信息（如有）

### 4. 使用限制
- 仅供个人学习使用
- 不用于商业用途
- 不大规模复制原作

## 版权检查清单

收集内容前确认：
- [ ] 评估版权风险等级（🟢低/🟡中/🔴高）
- [ ] 根据等级向用户发出相应提示
- [ ] 用户确认后继续收集
- [ ] 记录来源（URL、作者、日期）

## 个人使用场景

**所有内容均可收集，但需根据版权风险等级提示用户后继续。**

### 提示模板

**中风险提示：**
> ⚠️ 该内容可能受版权保护（来源：xxx）。建议仅供个人学习使用，不得用于商业用途或公开传播。是否继续？

**高风险提示：**
> ⚠️ 版权风险提示：该内容属于受版权保护的资料（来源：xxx）。收集后仅供个人学习使用，严禁商业用途和公开传播，可能面临法律风险。是否继续？

## 法律声明

本技能仅用于辅助学习资料收集，用户需对收集内容的版权合规性负责。涉及商业用途时建议咨询法律专业人士。

**用户在收到版权风险提示并确认继续后，需自行承担因版权问题可能产生的法律风险。**
